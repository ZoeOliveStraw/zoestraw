Krog Remake

Working with the MRC studio team, Krog is a remake of a mobile game from years ago. For Krog I wrote enemy AI, an object pooling system, the keyboard / controller remap system and some other smaller contributions to the game's code.

MRC Issue Tracker

Unity and Unreal add-on that allows users to send bug reports to project management software. I wrote code for sending web requests in editor and at run time. I also ported the software to VR in Unreal.

MRC XR Framework

I wrote code to create a package that contained prefabs and code to rapidly speed up the process of creating a VR game. Including basic functions like player locomotion, hand interaction, turning and putting it all in a rig that would be easy to work with. This project was designed to be used with a job training contract we would work on further down the line.
